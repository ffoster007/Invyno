## Get Started
```
git clone https://github.com/ffoster007/Invyno.git
cd Invyno
```
## For Linux/Macos Setup
```
cp webapp/.env.example webapp/.env
bash setup.sh
```
## Window Setup
```
copy webapp\.env.example webapp\.env
.\setup.ps1
```

## Build Desktop-App
## Linux/Macos
```
bash scripts/buildapp.sh
bash scripts/buildapp.sh dev
```

## Window
```
.\scripts\buildapp.ps1
.\scripts\buildapp.ps1 -Script dev
```
